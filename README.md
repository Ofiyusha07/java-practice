# Java Practice Assignments

This repository contains solutions to Java practice tasks.

## Description
All tasks are implemented in a single Java project.
Each task is placed in a separate Java class with a clear name.
The solutions focus on working with ArrayList and basic Java concepts.

## Topics covered
- ArrayList operations
- Searching and filtering elements
- Removing duplicates
- Working with strings
- Nested collections

## How to run
Open the project in IntelliJ IDEA.
Each task contains its own `main` method and can be run independently.
